## Smart-Attendance

* Smart-Attendance is a next-generation attendance management system that leverages AI to ensure accuracy and security. By combining Face Recognition and Voice Recognition, the platform provides a seamless, touchless, and fraud-proof way to track attendance for schools, offices, or events.
------------------------------
## 🚀 Key Features

* Dual-Biometric Auth: Uses both face and voice for multi-factor identity verification.
* Real-time Processing: Fast recognition with minimal latency.
* Anti-Spoofing: Prevents the use of photos or recordings to fake attendance.
* Automated Reports: Generates daily/monthly logs in CSV or PDF formats.
* User Dashboard: intuitive interface for both administrators and employees/students.

------------------------------
## 🛠️ Tech Stack

* Frontend: React.js / HTML5 / Tailwind CSS
* Backend: STREAMLIT
* AI/ML Libraries:
* OpenCV: For image processing and camera handling.
   * Dlib / Face_recognition: For high-accuracy facial feature mapping.
   * DeepSpeech / PyAudio: For voice capture and analysis.
* Database: SUPABASE

------------------------------

## 💡 How It Works

   1. Registration: Users register by capturing their face samples and a short voice clip.
   2. Verification: When marking attendance, the system scans the live face and asks for a voice passphrase.
   3. Matching: AI models compare the live input against the stored biometric templates.
   4. Logging: Upon a successful match, the database updates the timestamp for that specific user.
